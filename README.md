<h1 align="center">"Dulces"</h1>


<p align="center">
<img src="https://ssserch.com/Dulces/img/gumball.gif" alt="GIF de Dulces" width="300">
</p>

Un simple proyecto el cual te permite tomarte una foto para posteriormente escanearlo y deducir si es un disfraz de halloween o no.

Para eso necesitas colocar tu Token (de Google Gemini) manualmente, debido a que por defecto la pagina no lo tiene pre-configurado por razones de seguridad.

<i><small>Face-API no me pertenece (https://github.com/justadudewhohacks/face-api.js)</small></i>
